Assignment 02

SAAS Pricing Card Layout

CONCEPT

Build a pricing page for a fictional web-hosting SaaS product. Three plan cards sit side-by-side. The middle card is visually elevated as the recommended option. Above the cards, a toggle switches between Monthly and Annual billing.

SKILLS TESTED

Flexbox layout

CSS Custom Properties

Pseudo-elements

Box Shadow

Border Radius

Hover Transitions

Relative / em Units

Semantic HTML

WHAT MAKES IT TRICKY

The featured card must be scaled up without breaking the flex row alignment. The feature list inside each card uses flex column layout with a checkmark/dash icon, and the CTA button must be pinned to the card base with margin-top: auto.
