Assignment 01

Personal Bio Page

CONCEPT

Build a single-page personal portfolio site for a fictional developer. It includes a sticky nav, a hero section with an avatar, an About Me blurb, a skills chip row, a three-column project card grid, and a simple footer.

SKILLS TESTED

Semantic HTML

CSS Custom Properties

Flexbox

Box model

CSS variables

Hover states

Responsive Units

Google Fonts

WHAT MAKES IT TRICKY

Aligning the hero avatar + text side-by-side with flexbox while keeping the nav sticky. The three project cards must be equal-height flex items with the CTA button pinned to the bottom using margin-top: auto. Managing spacing entirely through CSS custom properties (no magic numbers).
